# 🔥 Firestack

## 📖 Getting started

### 📦 Install packages

`npm run install`

### 🚀 Start applications

`npm run server`

`npm run client`

### 🧪 Test application

Unit tests:

`npm run test`

E2E tests:

`npm run cypress`

Make sure server and client applications are running. To re-run the application, clear data in the [firestore emulator](http://localhost:4201/firestore/data/).

Select test suite(s).

### 🤬 Lint application

`npm run lint`

### 🔎 Analyze frontend bundle

`cd client`

`npm run analyze`

### 🔨 Kill port

`netstat -ano | findstr :PORT`

`taskkill /PID **** /F`
