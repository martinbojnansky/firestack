# 🔥 Firestack

## 📖 Getting started

### 📦 Install packages

> `npm i`

> `cd server`

> `npm i`

> `cd ../client`

> `npm i`

### 🚀 Start application

> `npm run start-server`

> `npm run start-client`

### 🧪 Test application

Unit tests:

> `npm run test-server`

> `npm run test-client`

E2E tests:

> `npm run cypress`

> Make sure server and client applications are running. To re-run the application, clear data in the [firestore emulator](http://localhost:4201/firestore/data/).

> Select test suite(s).

### 🤬 Lint application

> `npm run lint-server`

> `npm run lint-client`

### 🔎 Analyze bundle

> `npm run analyze`
